# Section 6: Gaps and Caveats
- Scope scanned: `6292` month-interval files under `data/historical/monthly/month/*/*.zip`; day-level files were only loaded for analyzed symbols (ADA, AVAX, BTC, DOGE, ETH, SOL, XRP).
- Bull-run detection is rule-based (local trough -> peak within 12 months, 30%+ gain). Different definitions can change interval boundaries.
- BTC dominance uses trade-value share proxy (`acc_trade_price`) instead of true market-cap dominance; interpret as attention/liquidity dominance, not supply-weighted dominance.
- CVD is proxied by candle direction x volume (`sign(close-open)*volume`), not tick-level aggressor flow.
- `pre_bull_score`, `btc_acc`, `stealth_cnt` were reconstructed from OHLCV-only proxies; these may differ from production feature engineering in code.
- [INFERRED] macro context tags (ETF cycle, post-halving narrative, alt-rotation regime) are explicitly inferred and not directly encoded in local dataset.
- Some symbols have shorter listing history; early runs may have limited 1-8 week lookback data, reducing sampled count and confidence.
