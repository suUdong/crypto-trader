2. **Logic Discovery**: 특정 종목이 오르기 직전의 수급 패턴을 수학적으로 추출한다.
3. **Coding**: 추출된 패턴을 'Scoring-based' 논리로 변환하여 Python 코드로 작성한다.
4. **Validation**: 3080으로 20회 이상의 파라미터 스윕을 돌려 '골든 매트릭스'를 완성한다.

- **Evidence First**: "감"이 아닌 "3080 데이터"로만 대화하라.
